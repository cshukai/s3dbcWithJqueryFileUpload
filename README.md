s3dbcWithJqueryFileUpload
=========================


A. dependencies:

(A) S3DB connector:

https://github.com/agrueneberg/S3DB-Connectivity

(B) JQuery File Upload:

https://github.com/blueimp/jQuery-File-Upload

B. Requirement:

(A) S3DB:

1. username
2. password
3. url of S3DB

(B) JQuery File Upload:

an input tag with the value of "type" attribute set as "file"
